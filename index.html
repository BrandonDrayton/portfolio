On-Play

OnPlay is a web application designed to provide relevant sports news and exclusive chat forums based upon a user's favorite team. This was built using React JS, Redux Toolkit, node JS, POSTGRESQL, and sequelize. I was responsible for the entire backend of this website as well as creating the global API slices for redux.

Coin Dash

The Coin Dash cryptocurrency application was designed to give you as much information as possible. Me-Tube

This project was designed to be a fully responsive youtube clone using React and MUI. The free API youtube-v31 was used to fetch data for videos, categories, and search functionality. You can search by name, category, or creator. The look and aesthetic of youtube was mimicked to give the best feel for a viewing experience.

Foodie-Bräu

Foodie-Bräu is a web application designed to search through different beers, add specific beers to a liked list, and to give food pairing recipes for a selected beer. Foodie-Bräu was created as part of a group front end project for the full time web development April 2022 cohort at Digital Crafts. The purpose of the project is to practice interacting with API databases, work with local storage, as well as html/css design and layout.
